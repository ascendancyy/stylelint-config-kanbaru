# [kanbaru] Personal stylelint config

## Install

```bash
$ npm install -D eslint @ascendancyy/stylelint-config-kanbaru
```

## Usage

In stylelint (.stylelintrc.*, stylelint.config.js):

```js
// stylelint.config.js
module.exports = {
  extends: ['@ascendancyy/stylelint-config-kanbaru'],
  // ...
};
```
